# Naše Kadaň

Informační portál o Kadani: zprávy, kulturní akce, historie města a praktické informace.

## Nasazení

Projekt je určený pro vlastní OVH VPS `57.129.43.215` a automatické nasazení přes self-hosted GitHub runner.

- kalendář akcí

- responzivní vzhled
